## 1. פונקציות *קולטות ומדפיסות* (ללא פרמטרים)

| סוג הפונקציה                               | הסבר בעברית                                                                  | C#                                                                                                                                           | הסבר ברוסית                                                          |
| ------------------------------------------ | ---------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------- |
| פונקציה שקולטת מספר ומדפיסה אותו           | הפונקציה **לא מקבלת פרמטרים**. היא בעצמה קולטת מספר מהמשתמש ואז מדפיסה אותו. | <pre><code class="language-csharp">
static void ReadAndPrintNumber()
{
    Console.Write("Enter a number: ");
    int num = int.Parse(Console.ReadLine());
    Console.WriteLine("You entered: " + num);
}
</code></pre> | Функция **считывает** число с клавиатуры и **выводит** его на экран. |
| פונקציה שקולטת שם ומברכת                   | שוב — בלי פרמטרים. הפונקציה שואלת מה השם ומדפיסה ברכה.                       | <pre><code class="language-csharp">
static void ReadNameAndGreet()
{
    Console.Write("Enter your name: ");
    string name = Console.ReadLine();
    Console.WriteLine("Hello, " + name + "!");
}
</code></pre> | Функция **считывает** имя пользователя и **показывает** приветствие. |
| פונקציה שקולטת שני מספרים ומדפיסה את הסכום | הפונקציה עושה גם קלט וגם פלט, בלי לקבל פרמטרים.                              | <pre><code class="language-csharp">
static void ReadTwoNumbersAndPrintSum()
{
    Console.Write("Enter first number: ");
    int a = int.Parse(Console.ReadLine());

    Console.Write("Enter second number: ");
    int b = int.Parse(Console.ReadLine());

    int sum = a + b;
    Console.WriteLine("Sum = " + sum);
}
</code></pre> | Функция **считывает** два числа и **выводит** их сумму.              |

> שימי/שימו לב: בכל שלושת הדוגמאות האלו אין פרמטרים בסוגריים, ואין ערך מוחזר (ההגדרה היא `void`).

## 2. פונקציות *שמקבלות ומחזירות* (עובדות עם נתונים, לא עם המשתמש)

| סוג הפונקציה                               | הסבר בעברית                                                                                     | C#                                                                                                                 | הסבר ברוסית                                                                            |
| ------------------------------------------ | ----------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------ | -------------------------------------------------------------------------------------- |
| פונקציה שמקבלת שני מספרים ומחזירה את הסכום | הפונקציה **מקבלת** שני פרמטרים ומחזירה תוצאה. אין בה `Console.ReadLine` או `Console.WriteLine`. | <pre><code class="language-csharp">
static int Add(int a, int b)
{
    int sum = a + b;
    return sum;
}
// דוגמת שימוש:
// int s = Add(3, 5);
</code></pre> | Функция **получает** два параметра `a` и `b` и **возвращает** их сумму.                |
| פונקציה שבודקת אם מספר חיובי               | מקבלת מספר אחד ומחזירה `true` או `false`.                                                       | <pre><code class="language-csharp">
static bool IsPositive(int num)
{
    return num > 0;
}
// דוגמת שימוש:
// bool ok = IsPositive(7);
</code></pre> | Функция **получает** число и **возвращает** логическое значение: положительное ли оно. |
| פונקציה שמחזירה ממוצע של שני מספרים        | מקבלת שני מספרים ממשיים (double) ומחזירה את הממוצע שלהם.                                        | <pre><code class="language-csharp">
static double Average(double x, double y)
{
    double avg = (x + y) / 2.0;
    return avg;
}
// דוגמת שימוש:
// double a = Average(4.0, 10.0);
</code></pre> | Функция **принимает** два числа типа `double` и **возвращает** их среднее значение.    |

## הדגשה פדגוגית (לטקסט של הצ’אט / המסמך)

* **פונקציות שקולטות ומדפיסות**
  * עברית: *קולטת*, *מדפיסה*
  * רוסית: **считывает**, **выводит** (נרדפות: *вводит данные*, *печатает*, *показывает на экран*)
  * הן מדברות עם המשתמש (קלט/פלט), לא מקבלות פרמטרים, לא מחזירות ערך.

* **פונקציות שמקבלות ומחזירות**
  * עברית: *מקבלת*, *מחזירה*
  * רוסית: **получает**, **возвращает** (נרדפות: *принимает параметры*, *выдаёт результат*)
  * הן *לא* עושות קלט/פלט בעצמן – הן עובדות על נתונים שנמסרו להן, ומחזירות תשובה למי שקרא להן.

אם תרצה, בשלב הבא אפשר להוסיף גם טבלה של “שאלות תרגול” לתלמידים: לזהות לפי קוד אם זו פונקציה מסוג *קולטת ומדפיסה* או *מקבלת ומחזירה*, ולהסביר ברוסית ובעברית.
